# Darke Chess Analysis Tool

A desktop application for real-time chess analysis using screen capture and Stockfish engine integration.

## Features
- Screen overlay for chessboard selection.
- Real-time board state recognition.
- Move recommendation using Stockfish.
- Customizable engine strength to simulate human play.

## Usage
1. Run `python main.py`
2. Click "Select Area" to define the chessboard region.
3. Start analysis.

## Disclaimer
This tool is for educational purposes only. Do not use it to cheat in online games.
